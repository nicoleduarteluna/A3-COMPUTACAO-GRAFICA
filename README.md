# COMPUTACAO-GRAFICA-A3-FLORES
Repositório para a entrega do projeto A3 da UC de computação gráfica dos integrantes citados abaixo.
O objetivo deste trabalho é possibilitar que o usuário insira uma chave de busca personalizada, e após isso o sistema criará uma imagem exclusiva contendo várias flores em posições e cores aleatórias.

#### Como rodar?

Abrir o arquivo index.html com liveServer pelo VSCode.
Cada identificador digitado irá gerar uma chave hash que poderá ser visualizada a partir do botão ao lado.
